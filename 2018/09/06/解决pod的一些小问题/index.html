解决pod的一些小问题

好了最近状态有些不太好。。昨天晚上到今天（有点感冒身体不舒服.png）好了 昨天晚上在创建项目的时候用了下pod init结果 Oh no, an error occurred.
好吧。。很难受。。现在修复下吧那。
怎么搞呢 当然是百度 先看一下error 信息

RuntimeError - [Xcodeproj] Unknown object version.
/Users/kys-20/.rvm/gems/ruby-2.4.0@global/gems/xcodeproj-1.5.3/lib/xcodeproj/project.rb:217:in `initialize_from_file'

简单查了一下说是这个Xcode版本和pod的版本不一样，好吧 很简单粗暴的解决下

sudo gem install cocoapods --pre
更新下pod
结果还是不行 好吧 在更新下gem好了 
gem update
再次pod init
ok

这里要说一下 关于gem更换源的问题现在网上的很多教程都是要更换gem的源更换成
https://gems.ruby-china.org
但是这网站换了域名了,换成了https://gems.ruby-china.com 这个这个大家注意下 其实看报错信息可看的到的，就是提醒下大家。

这里为小伙伴再说下pod 安装吧

  1. sudo gem install cocoapods （可以考虑更新下gem）
  2. 没了

Mac 都是自带gem 的更新下 换下源就OK了
