关于Git的简单使用方式

首先在Git仓库创建一个远程仓库（国内有许多的平台，例如码云，Coding）当然还有著名的Github（同性交友网站） 书归正传 简单说下Git的使用我这个了用的平台是Coding和小伙伴一起用的 我先学会了 所有写点东西记录下也帮助他快速理解下。

首先

在创建了远程仓库之后的话，我们需要在本地也去创建一个本地的仓库，我用的Mac电脑所有就直接 cd 到一个新建的文件夹（准备放本地仓库的地方） 然后执行命令 git clone url

提交上传代码

OK 然后就进入项目开始改代码吧，随便改，改完后后会发现Xcode文件上面有些标记比如A ， M ？ 之类的这个意思的话可以去查下，简单的说就是标记你改了那个文件。
anyWay 我们改完了 不能就这样放在本地吧，我们肯定是要提交到云端然后多人协作开发的。

git add .

这个命令是将你修改了的文件都添加都了本地仓库的暂存区（具体的这些概念我也不是很理解 建议可以看看廖雪峰的git教程）

git commit -m “文字”

这个命令是将你改的文件交至HEAD所指向的分支，暂存区的文件将回到未修改状态
文字指的就是注释了，就是你本次提交干了啥。

git status （查看你改了啥）

这个命令会显示你改了 啥了 这里就不截图了。。。懒.png

git pull

这个命令啊就是去你的远程仓库去拿数据看看给你的本地仓库更新下，多人开发的话 要用下的。

git push

这个命令就是提交咯 然后就去看看你的成果吧

总结
